Clod

AI-powered Twitter bot that responds to mentions with wit and intelligence, powered by Claude.

What is Clod?

Clod is an autonomous Twitter bot that monitors mentions and generates contextual, engaging responses using Claude AI. It combines the power of large language models with robust engineering practices to deliver reliable, entertaining interactions.

Features

Claude AI Integration

Powered by Claude Sonnet for intelligent, context-aware responses that feel natural and engaging.

Fault Tolerant

Circuit breaker pattern, automatic retries, and graceful degradation keep the bot running smoothly.

Rate Limit Aware

Smart handling of Twitter API limits with exponential backoff and request throttling.

Production Ready

Comprehensive metrics, health checks, and graceful shutdown for reliable deployment.

Quick Start

# Clone the repository
git clone https://github.com/clodlive/clod.git
cd clod

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env

# Run the bot
python bot.py
